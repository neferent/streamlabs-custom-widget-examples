⭐ subscription Twitch YouTube

New sub, resub, and gifted subs all share type: 'subscription'. Use gifted and months to distinguish them.

FieldTypeNotes
namestringSubscriber's name (or gift recipient's name)
monthsnumberCumulative months subscribed. 1 for a new sub
messagestringShare message — may be empty
giftedbooleantrue if this sub was a gift
gifterstringName of the person who sent the gift — only present when gifted: true
amountnumberOn gift bombs: number of subs gifted in one go

💰 donation All platforms

Streamlabs tip / donation. Also used by charity integrations — check platform to distinguish the source.

Heads up: amount may be 0 on real donations due to a known Streamlabs bug. Always use formattedAmount for display, and amount only for numeric comparisons where you've verified it works.
FieldTypeNotes
namestringDonor's name
amountnumberRaw numeric value — may be 0 on real events (see note)
formattedAmountstringCurrency-formatted string e.g. '$25.00' — use this for display
currencystringISO currency code e.g. 'USD'
messagestringDonor's message — may be empty
 
document.dispatchEvent(new CustomEvent('onEventReceived', { detail: {
  type: 'donation', name: 'TestUser', from: 'TestUser',
  amount: 25, formattedAmount: '$25.00', currency: 'USD',
  message: 'Great stream!', isTest: true,
}}));

💬 Chat messages Twitch

Twitch chat messages arrive through onEventReceived with command: 'PRIVMSG' and type: 'chatmessage'. Always filter on both.

FieldTypeNotes
fromstringLowercase username e.g. 'neferent'
bodystringMessage text
ownerbooleantrue if the sender is the channel owner / broadcaster
subscriberbooleantrue if sender is a subscriber
userTypestring'user' | 'mod' | 'vip'
tags['mod']string'1' if moderator, '0' if not — a string, not a boolean
tags['display-name']stringDisplay name with original capitalisation
tags['color']stringUser's chat colour — empty string if unset
messageIdstringUnique message ID
 
// Standard pattern for chat command handling
document.addEventListener('onEventReceived', (obj) => {
  const d = obj.detail;
  if (d.command !== 'PRIVMSG' || d.type !== 'chatmessage') return;

  const sender  = (d.from || '').toLowerCase();
  const message = (d.body || '').trim().toLowerCase();
  const isMod   = d.tags['mod'] === '1';
  const isOwner = d.owner === true;
});
// Fake chat message for local testing
document.dispatchEvent(new CustomEvent('onEventReceived', { detail: {
  type: 'chatmessage', command: 'PRIVMSG',
  from: 'testuser', body: '!death',
  owner: false, subscriber: false, userType: 'mod',
  tags: { 'display-name': 'TestUser', 'mod': '1', 'color': '#FF0000' },
}}));
