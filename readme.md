# Streamlabs Custom Widget Examples

A collection of finished, ready-to-use examples for Streamlabs' Custom Widget — a widget type that accepts 100% custom HTML, CSS, and JS with no restrictions on structure or behaviour.

Visit the **[project page](https://neferent.github.io/streamlabs-custom-widget-examples/)** for a full overview and download link.

## How to use

Each example lives in its own folder. In Streamlabs, open a Custom Widget and paste:

- Everything inside `<body>` → **Custom HTML**
- Contents of the `.css` file → **Custom CSS**
- Contents of the `.js` file → **Custom JS**

Preview locally using [Phoenix Code](https://phcode.io/).
